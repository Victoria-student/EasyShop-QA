# EasyShop QA Project

## О проекте

EasyShop — учебный проект интернет-магазина электроники, созданный для получения практического опыта ручного тестирования.

Проект включает полный цикл QA:

- Requirements
- Test Plan
- Check Lists
- Test Cases
- Bug Reports
- Retesting
- Regression Testing
- Release Notes

---

## Цель проекта

Получить практический опыт работы Manual QA максимально приближенный к коммерческой разработке.

В рамках проекта самостоятельно выполняется:

- анализ требований;
- разработка тестовой документации;
- проведение функционального тестирования;
- оформление баг-репортов;
- проведение ретестов;
- регрессионное тестирование после исправлений.

## Инфраструктура

Проект является самописным учебным веб-приложением.

Сайт развернут на виртуальном хостинге REG.RU.

Доступ к файлам проекта осуществляется по FTP с использованием FileZilla.

Все изменения HTML, CSS и JavaScript публикуются непосредственно на тестовом сервере.
