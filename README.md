# EasyShop QA Project

## О проекте

EasyShop — учебный проект интернет-магазина, созданный для практики ручного тестирования.

В рамках проекта выполняются:

- Test Planning
- Test Cases
- Check Lists
- Bug Reporting
- Regression Testing
- Retesting
